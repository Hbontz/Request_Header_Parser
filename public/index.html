Request Header Parser
Usage

Add /api/whoami to the end of this page's URL to see your IP address, the language you are using, and your computer's operating system.

Example Input
http://a-request-header-parser.herokuapp.com/api/whoami
Example Output
{ ip_address: 11.111.111.111, language: "en-US", os: "X11; Linux x86_64" }
